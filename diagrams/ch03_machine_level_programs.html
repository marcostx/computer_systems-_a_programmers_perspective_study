IA32 General-Purpose Registers

8 general-purpose registers. Each 32-bit register can be accessed as 16-bit or 8-bit sub-registers. Click to highlight.

x86-64 Register Set

Extended to 16 general-purpose 64-bit registers. Backward compatible with IA32 sub-register naming.

Key differences from IA32:
- 16 registers instead of 8 (r8-r15 are new)
- All 64 bits wide (%rax, %rbx, etc.)
- %rsp is the stack pointer (no frame pointer by default)
- First 6 integer args passed in registers: %rdi, %rsi, %rdx, %rcx, %r8, %r9
- Return value in %rax

See how C control structures translate to assembly using condition codes and jumps.

Condition Codes

C Code

if (x < y) { result = x; } else { result = y; }

Assembly (IA32)

cmpl %edx, %eax ; compare x, y jge .L2 ; if x >= y, goto L2 movl %eax, %ecx ; result = x jmp .L3 ; skip else .L2: movl %edx, %ecx ; result = y .L3:

while loop (C)

while (i < n) { sum += arr[i]; i++; }

Translated (jump-to-middle)

jmp .test ; goto test .loop: addl (%edx,%ecx,4), %eax addl $1, %ecx ; i++ .test: cmpl %ebx, %ecx ; i < n ? jl .loop ; if yes, loop

Switch uses a Jump Table

The compiler creates a jump table (array of code addresses) for switch statements. The switch variable is used as an index into this table, providing O(1) dispatch regardless of the number of cases.

; Jump table at .L4: .L4: .long .L3 ; case 0 .long .L2 ; case 1 .long .L5 ; case 2 .long .L6 ; case 3 ; Dispatch: cmpl $3, %eax ; if n > 3 ja .default ; goto default jmp *.L4(,%eax,4) ; goto table[n]
