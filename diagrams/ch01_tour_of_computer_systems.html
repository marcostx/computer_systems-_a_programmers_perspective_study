Concurrency and Parallelism

Three levels of parallelism exploited by modern systems.

Thread-Level Parallelism

Multiple execution flows within a single process or across processes. Multi-core processors can execute multiple threads truly in parallel.

Core 0: Thread A
Core 1: Thread B
Core 2: Thread C
Core 3: Thread D

Instruction-Level Parallelism (ILP)

Modern processors execute multiple instructions simultaneously using pipelining and superscalar execution. Can sustain rates of 2-4 instructions per clock cycle.

Cycle 1: Fetch I1, Decode I0
Cycle 2: Fetch I2, Decode I1, Exec I0
Cycle 3: Fetch I3, Decode I2, Exec I1, Mem I0

SIMD Parallelism

Single Instruction, Multiple Data. Special hardware that allows a single instruction to cause multiple operations in parallel, mainly for image/video/audio processing.

ADDPS xmm0, xmm1
// Adds 4 float pairs
// simultaneously in one instruction
