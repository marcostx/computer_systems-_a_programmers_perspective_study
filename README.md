# Computer Systems: A Programmer's Perspective — Study Materials

Study companion for **CS:APP** (Bryant & O'Hallaron, 2nd Edition), containing the textbook, interactive diagrams, and self-assessment quizzes covering all 12 chapters.

## Contents

- **Book** — Full textbook PDF (2nd Edition)
- **Diagrams** (`diagrams/`) — Interactive HTML diagrams for each chapter, visualizing key concepts such as memory hierarchy, processor pipelines, virtual memory layout, and more
- **Quizzes** (`quizzes/`) — Multiple-choice quizzes with separate answer keys, split into two sets:
  - Chapters 1–6: Tour of Computer Systems, Representing Information, Machine-Level Programs, Processor Architecture, Optimizing Performance, Memory Hierarchy
  - Chapters 7–12: Linking, Exceptional Control Flow, Virtual Memory, System-Level I/O, Network Programming, Concurrent Programming
